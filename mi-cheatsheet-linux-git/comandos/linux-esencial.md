# Comandos Linux Esenciales

## Navegación
| Comando | Descripción | Ejemplo |
|---------|-------------|---------|
| `pwd` | Muestra directorio actual | `pwd` |
| `ls` | Lista archivos | `ls -la` |
| `cd` | Cambia directorio | `cd /home` |

## Archivos
| Comando | Descripción | Ejemplo |
|---------|-------------|---------|
| `touch` | Crea archivo vacío | `touch archivo.txt` |
| `cp` | Copia archivo | `cp origen destino` |
| `mv` | Mueve/renombra | `mv viejo.txt nuevo.txt` |
| `rm` | Borra archivo | `rm archivo.txt` |

## Permisos
| Comando | Descripción | Ejemplo |
|---------|-------------|---------|
| `chmod` | Cambia permisos | `chmod 755 script.sh` |
| `ls -la` | Ver permisos | `ls -la` |

## Búsqueda
| Comando | Descripción | Ejemplo |
|---------|-------------|---------|
| `find` | Busca archivos | `find . -name "*.txt"` |
| `grep` | Busca texto | `grep "palabra" archivo` |
